# Getting started

1. Fork this repo
2. Clone your fork
3. Install node on your machine
4. Run `npm install` from within the project

*For bootstrap/theme documentation run `gulp docs`*
*To compile run `gulp`*
*To deploy to github pages run `gulp deploy`*
